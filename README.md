## Problem1

### AreaSplit.cpp

用于将一个凸多边形的搜索范围等面积划分为若干区域

#### 输入

1. 输入凸多边形顶点个数`nPoints`和划分区域个数`nAreas`
2. 输入无人机飞行速度`V`和搜索半径`d`
3. 输入`nPoints`个点的坐标，形式为`x y`

 
#### 输出

搜索用时`t`
